Having a Virtual Enviornment is NECESSARY:-
Creation:- (ALWAYS TRY INSTALLING TABPY IN THE VIRTUAL ENVIORNMENT)
1. Go to you project folder:-
Open command prompt and run:-
cd "C:\Users\'YourUserName'\Downloads\YourProjectFolderName"
.
2. Now create a virtual Enviornment:- (ALWAYS TRY INSTALLING TABPY IN THE VIRTUAL ENVIORNMENT)
python -m venv .venv
.
3. Activate the virtual environment:-
.venv\Scripts\activate
After the activation terminal should show something like:-
(.venv) C:\Users\YourUserName\Downloads\YourProjectFolderName>
.
4. Upgrade pip:-
python -m pip install --upgrade pip
.
5. Install the required packages:- (ALWAYS TRY INSTALLING TABPY IN THE VIRTUAL ENVIORNMENT)
python -m pip install pandas tabpy
.
6. I am creating a requirements.txt file too so that it becomes easy to install pre-requisites:-
pip freeze > requirements.txt
To install em, just run:- pip install -r requirements.txt
.
7. Check the installed packages:-
python -m pip show pandas
python -m pip show tabpy
.
8. Start tabpy:-
In the same activated virtual environment, run:-
tabpy
Or if your tabpy is installed in some other location (ALWAYS TRY INSTALLING TABPY IN THE VIRTUAL ENVIORNMENT), run:-
"C:\Users\Vikrant Singh Thakur\PycharmProjects\PythonWorks\.venv\Scripts\tabpy.exe"
.
If TabPy starts correctly, the terminal will show several log messages. The important point is that TabPy should be running on:

localhost
port 9004

This means Tableau Prep can connect to Python using:

Server: localhost
Port: 9004
.
